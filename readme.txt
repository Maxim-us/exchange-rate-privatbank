Плагин позволяет вывести курс валют от Приват Банка на сайте в области виджетов.

== Description ==

Плагин позволяет на основе PrivatBank API вывести актуальный курс валют от Приват Банка на сайте в области виджетов.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'Курс Валют Приват банк'
3. Activate the plugin from your Plugins page.

= From WordPress.org =

1. Download 'Курс Валют Приват банк'.
2. Upload the 'WP Plugin Skeleton' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate 'Курс Валют Приват банк' from your Plugins page.
